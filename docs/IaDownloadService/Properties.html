IA_MAX_REDIRECTS =

Maximum length of redirection chain.

In the case of IA downloads, this includes both redirections driven by the HTTP redirects and variations attempted within #transmit in order to find a suitable fall-back download format.

IA_COOKIES =

Cookies to be sent to the IA server.

Implementation Notes

These values were obtained from a desktop development VM after installing the “ia” Python script and running “ia configure” with the Email address “emmadso@bookshare.org”. This generates a configuration file ~/.ia which contains an “[s3]” section with the S3 access key and secret, and a “[cookies]” section which contains these values.

See Also:

  • https://archive.org/services/docs/api/internetarchive/quickstart.html#configuring
