PATTERN =

Valid values for this type match this pattern one of these patterns.

{
  year:  /^\d{4}$/,
  day:   /^\d{4}-\d\d-\d\d$/,
  exact: /^\d{4}-\d\d-\d\dT\d\d:\d\d:\d\dTZD$/
}.freeze
