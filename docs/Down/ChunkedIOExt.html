Overview

These additions make ChunkedIO look sufficiently “IO-like” to IOWindow (defined by the 'archive-zip' gem) so that files uploaded to AWS S3 can be processed by Archive::Zip::Entry#parse.

Implementation Notes

ChunkedIO does not care about 'lineno' (and neither does anything else in the call chain of interest) so providing a dummy instance variable should be harmless.
