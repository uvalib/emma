Overview

Bs::Record::AssignedTitleMetadataSummary

Implementation Notes

Similar to Bs::Record::TitleMetadataSummary, but without these fields:

:arrangers
:contentWarnings
:contributors
:lyricists
:translators

And adds these fields:

:assignedBy
:dateAdded
:dateDownloaded

See Also:

  • https://apidocs.bookshare.org/reference/index.html#_assigned_title_metadata_summary
