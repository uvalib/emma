POST /recordGets

Retrieves one or more metadataRecords in the search index. Records are uniquely identified by the emma_repository, emma_repositoryRecordId, dc_format, and (optionally) emma_formatVersion.

The number of records to be retrieved at once is capped at 1000.

Variations

HTTP response codes

200 OK Items retrieved. 400 Bad Request Invalid input.
